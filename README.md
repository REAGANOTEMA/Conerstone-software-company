# NextERP Systems

Enterprise Management Ecosystem - A comprehensive business management platform built for modern organizations.

## Features

### Core Business Management
- **Dashboard**: Real-time analytics and business insights
- **Projects**: Project management with Kanban boards and tracking
- **HR**: Human resources management and employee tracking
- **Finance**: Financial reporting and budget management
- **Marketing**: Campaign management and analytics
- **Compliance**: Legal document management and regulatory compliance

### Educational Portal
- **Academic Management**: Course management and degree tracking
- **Virtual Classroom**: Live streaming classes with interactive features
- **Assignments**: AI-powered assignment management and grading
- **Grades**: Comprehensive grade tracking and transcripts
- **Community**: Student and instructor collaboration tools

### Advanced Features
- **Real-time Notifications**: Multi-channel notification system
- **Security**: Advanced authentication and access control
- **Billing**: Subscription management and payment processing
- **Multi-tenant Architecture**: Scalable for multiple organizations

## Technology Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS
- **UI Components**: Radix UI, Shadcn/ui
- **State Management**: React Hooks, Context API
- **Routing**: React Router v6
- **Build Tool**: Vite
- **Icons**: Lucide React

## Getting Started

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
4. Open your browser and navigate to `http://localhost:5173`
